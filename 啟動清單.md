# Supervisor 啟動清單

本專案需要在 Supervisor 上設定以下 **2 個** 後端常駐服務。

> Redis 和 MySQL 由系統 / 寶塔面板管理，不需另外設定 Supervisor。

---

## 1. FastAPI（Gunicorn + Uvicorn）

| 項目 | 值 |
|---|---|
| 名稱 | `suricata-api` |
| 工作目錄 | `/www/wwwroot/web-suricata-test` |
| 執行命令 | 見下方 |
| 日誌目錄 | `/www/wwwlogs/` |

```bash
/www/wwwroot/web-suricata-test/.venv/bin/gunicorn app.main:app \
    -w 4 \
    -k uvicorn.workers.UvicornWorker \
    --bind 127.0.0.1:8000 \
    --access-logfile - \
    --error-logfile -
```

---

## 2. Celery Worker

| 項目 | 值 |
|---|---|
| 名稱 | `suricata-worker` |
| 工作目錄 | `/www/wwwroot/web-suricata-test` |
| 執行命令 | 見下方 |
| 佇列 | `analysis` |
| 並行數 | `2` |

```bash
/www/wwwroot/web-suricata-test/.venv/bin/celery \
    -A app.workers.celery_app worker \
    --concurrency=2 \
    -Q analysis \
    --loglevel=info
```

---

## 寶塔 Supervisor 設定步驟

1. 進入寶塔面板 → **軟體商店** → 搜尋 **Supervisor** → 安裝
2. 進入 Supervisor 管理介面 → **新增守護進程**
3. 分別新增上述 2 個服務，填入：
   - **名稱**：如 `suricata-api`
   - **啟動命令**：上方完整指令（合成一行）
   - **執行目錄**：`/www/wwwroot/web-suricata-test`
   - **進程數量**：`1`（Gunicorn 內部已有 4 workers）
4. 儲存後點擊「啟動」

### 合成一行版本（方便貼入寶塔）

**suricata-api：**
```
/www/wwwroot/web-suricata-test/.venv/bin/gunicorn app.main:app -w 4 -k uvicorn.workers.UvicornWorker --bind 127.0.0.1:8000 --access-logfile - --error-logfile -
```

**suricata-worker：**
```
/www/wwwroot/web-suricata-test/.venv/bin/celery -A app.workers.celery_app worker --concurrency=2 -Q analysis --loglevel=info
```

## 前置條件確認

啟動前請確認以下服務已正常運行：

- [x] **MySQL** — 寶塔資料庫管理中正常運行
- [x] **Redis** — `redis-cli ping` 回應 `PONG`
- [x] **NAS 掛載** — `ls /mnt/nas` 可看到資料夾
- [x] **資料庫遷移** — 已執行過 `alembic upgrade head`
- [x] **管理員帳號** — 已建立 admin 使用者
